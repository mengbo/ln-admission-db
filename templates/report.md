## 1. 定位分析

### 等效分换算表

| 年份 | 分数 | 位次 | 说明 |
|---|---|---|---|
| 今年（如 2026） | 考生原始分 | 考生位次 | 直接给出 |
| 去年（如 2025） | 去年等效分 | 考生位次（与今年相同） | 用考生位次反查去年 rank 表 |
| 前年（如 2024） | 前年等效分 | 考生位次（与今年相同） | 用考生位次反查前年 rank 表 |

> AI 输出 1-2 段趋势解读（分数走势、竞争激烈程度、近年波动），用**家长能懂的语言**。
> 不要解释换算原理、SQL、位次反查等技术细节。
>
> **降级**：分数缺失则本节跳过；去年或前年 rank 表缺失则只展示有的年份。

> **评分说明**：AI 根据考生情况（分数、选科、偏好）逐一查看每个候选志愿后凭常识给出综合分（0-10 分），反映"该志愿对考生的推荐价值"。综合分高 = AI 推荐度强，不代表录取概率。
>
> **排序**：按代表位次升序排列（位次小 = 门槛高 = 排前），这是辽宁"分数优先、遵循志愿"检索逻辑决定的填报顺序。家长可按本表顺序挑出 112 个填入辽宁志愿系统。

| 序号 | 代表位次 | 院校 | 专业 | 计划数（今年） | 前年（如 2024）分/位次 | 去年（如 2025）分/位次 | 综合分 | 评语 |
|---|---|---|---|---|---|---|---|---|
| 1 | | | | | | | | |
| 2 | | | | | | | | |
| ... | | | | | | | | |
| 200 | | | | | | | | |

**列说明**：
- **代表位次**：去年与前年录取位次中较小的一个（两年都有数据时）。位次越小门槛越高
- **前年/去年分/位次**：两年原始数据已直接体现波动趋势
- **综合分**：AI 评分（0-10），仅在本推荐表内有效
- **评语**：AI 对该志愿的简要点评（含学校层次、专业特色、适合程度等）

> 完整的 ~1000 行评分大表见同目录 CSV 文件。
