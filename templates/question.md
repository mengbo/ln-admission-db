# 考生信息问卷

## 使用方法

1. 复制下方"考生信息"开始到文件结尾的内容到对话中。
2. 在每行冒号后面填上对应信息；留空 = 用括号里的缺省值。
3. 标注"必填"的项必须填，其余均可留空。
4. 辽宁平行志愿无调剂问题。

考生信息
分数（必填）：
首选科目（物理/历史 缺省物理）：
选科组合（如 化学+生物 必填）：
想学专业（缺省不限）：
不想学的专业（缺省无）：
升学或就业方向（缺省未定；常见：本科就业/读研/出国/考公/央国企）：
学校位置（缺省 600+ 全国不限，600- 倾向辽宁周边；可填具体省份/城市）：

其他（缺省跳过；可写身体条件/特殊要求等）：
