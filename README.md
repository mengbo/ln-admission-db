# ln-admission-db

辽宁省高考招生本地化数据库 + AI Agent 辅助报考工具包。

- 📊 **本地离线**：2022–2026 招生计划、录取分数、一分一段表打包进单个 SQLite 文件，零网络依赖。
- 🤖 **AI 辅助**：配合 [OpenCode](https://opencode.ai) / Claude Code 等 Agent，用自然语言提问即可自动跑 SQL、生成冲稳保清单。
- 📝 **完整志愿指南**：按 6 轮对话逐轮向 AI 提问，最终输出可打印的志愿表。

![OpenCode 报考助力示例](https://github.com/mengbo/ln-admission-db/blob/main/assets/screenshot_oc.png?raw=true)

## 🎯 给家长看：这是啥、能干啥

高考志愿填报的核心问题只有两个：**我这个分能上啥学校？** **哪个学校哪个专业最适合我家孩子？**

本项目把辽宁省 5 年的招生计划、录取分数、一分一段表打包成一个 SQLite 数据库，再搭配 AI 助手（OpenCode / Claude Code）就能：

- 查分数对应的全省位次（一分一段表）
- 算今年分数相当于往年的多少分（等效分）
- 按"冲、稳、保"三档生成院校 + 专业清单
- 过滤地域、专业方向、家庭经济承受范围等个性化条件
- 最终输出一份**完整的、可打印的志愿填报指南**

你不需要懂 SQL，只需要会跟 AI 助手说话就行。

## 🛠 给开发者看：背后原理

整个数据库围绕**三张表**展开：

| 表 | 作用 | 关键字段 |
|---|---|---|
| `plan_YYYY` | 院校专业的招生计划 | `inst`, `major`, `plan_num`, `first`, `second` |
| `score_phyYYYY` / `score_hisYYYY` | 院校专业的最低录取分 | `inst`, `major`, `min_score` |
| `rank_phyYYYY` / `rank_hisYYYY` | 一分一段表（分数→位次） | `score`, `rank` |

核心查询模式：

```sql
-- 2024 物理类 620 分对应的位次
SELECT score, rank FROM rank_phy2024 WHERE score = 620;

-- 2024 物理类 605~635 分的院校专业
SELECT s.inst, s.major, s.min_score, r.rank
FROM plan_2024 p
JOIN score_phy2024 s ON p.inst_code = s.inst_code AND p.major_code = s.major_code
JOIN rank_phy2024    r ON s.min_score = r.score
WHERE p.first LIKE '%物%' AND s.min_score BETWEEN 605 AND 635
ORDER BY s.min_score DESC;
```

详细字段说明见 [docs/db_schema.md](./docs/db_schema.md)。

## 🤖 AI Agent 辅助报考：3 步上手

> 强烈推荐 [OpenCode](https://opencode.ai) 或 Claude Code：本地 shell 即可调 `sqlite3`，零配置。  
> 不推荐使用 MCP 协议方案（Cline / Cherry Studio），相关文档仅作历史参考：[docs/MCP.md](./docs/MCP.md)。

1. 在项目根目录启动 OpenCode / Claude Code。
2. 让 Agent 执行 `./scripts/createdb.sh` 构建数据库。
3. 按下方"考生信息清单"整理思路，再向 Agent 提问（模板见"提示词模板库"）。

## 🚩 使用前必读：考生信息清单

向 AI 提问前，**先把考生情况整理清楚**——这些信息会直接决定志愿表长什么样。

### 1. 基本信息
- 高考分数、首选科目（物理 / 历史）、选科组合
- 对应的一分一段位次（让 Agent 帮你查）
- 是否艺体生、少数民族、是否参与特殊招生

### 2. 专业好恶
- **喜欢**：例如计算机、电子信息、临床医学、师范类、电气工程
- **排斥**：例如不学数学、不学化学、不能接受临床 / 解剖 / 动物实验
- **不冷不热**：例如"可以接受但别太多"的方向

### 3. 院校地理位置
- 省份偏好（必须省内 / 南方 / 北方 / 不限）
- 城市级别（一线 / 新一线 / 省会 / 地级市）
- 离家距离、气候适应（南方孩子怕冷、北方孩子怕潮湿）

### 4. 家庭背景与社会关系
- 父母行业可对接的资源（医生 → 医学院；公检法 → 法学；建筑 → 土木；银行 → 财经）
- 体制内 vs 体制外偏好（影响考公、央国企导向）
- 是否有家族企业、长期合作单位（影响专业选择）

### 5. 经济与学制
- 学费承受区间：普通公办 ≤ 6000 / 中外合作 1.5–10 万 / 民办 2–5 万
- 是否接受民族班、预科班、定向招生
- 本科直接就业 vs 读研 vs 出国

### 6. 就业偏好
- 目标方向：考公、央国企、私企、互联网、外企、创业
- 目标行业：IT、金融、医疗、教育、制造、能源
- 是否有"铁饭碗"刚需

### 7. 身体与特殊条件
- 视力、色觉（部分专业受限）
- 性别（部分专业有性别限制或男女比例要求）
- 是否服从专业调剂（**强烈影响录取概率**，需要明确）

## 💬 提示词模板库

直接复制下方任一条发给 AI Agent 即可。**[模板 5]** 是完整的志愿填报对话流程，建议完整跑一遍。

### 模板 1：等效分换算

> 我家孩子 2026 年辽宁物理类考了 **XXX 分**，请帮我：
> 1. 查 2026 物理类一分一段表，得到位次。
> 2. 用这个位次，分别回查 2025、2024、2023 物理类一分一段表，得到等效分。
> 3. 整理成表格输出。

**期望输出**：一张 3 列表（年份、等效分、与今年的分差），含趋势解读。

### 模板 2：冲稳保院校清单

> 我家孩子 2026 物理类 **XXX 分、位次 XXX**，请生成冲稳保三档院校清单：
> - 冲：等效分 + 15 分以上
> - 稳：等效分 ± 15 分
> - 保：等效分 - 15 分以下
>
> 每条标注：院校名、专业名、计划数、近三年最低分与位次、所在地、学费档次。

**期望输出**：冲 / 稳 / 保三段，每段一张表（按分数降序）。

### 模板 3：单所院校深度分析

> 请深度分析 **XXX 大学** 在辽宁物理类近 3 年（2024 / 2025 / 2026）的招生数据：
> 1. 哪些专业在辽宁招生、各专业计划数。
> 2. 各专业近三年最低录取分与对应位次。
> 3. 是否有新设专业、停招专业、计划数大幅变化。
> 4. 在我家孩子位次附近的录取概率评估。

**期望输出**：4 段分析 + 关键数据表 + 概率结论（冲 / 稳 / 保）。

### 模板 4：单类专业横向对比

> 我家孩子想读 **计算机 / 电子信息** 类，请列出辽宁物理类 2026 年所有招收此类专业的院校：
> 1. 院校层次（985 / 211 / 双一流 / 普通本科）。
> 2. 各院校此专业的计划数。
> 3. 各院校此专业 2024–2025 最低分与位次。
> 4. 按位次排序，标注稳 / 冲建议。

**期望输出**：一张大表（按 2025 位次升序）+ 前 5 / 后 5 简要点评。

### 模板 5：完整志愿填报指南（按 6 轮对话逐步生成）

按以下 6 轮**逐轮**向 AI 提问，每轮确认输出后再进下一轮。

**第 1 轮｜等效分定位**  
> 孩子 2026 物理类 **XXX 分**。先查 2026 物理类位次，再算 2024 / 2025 等效分，整理成表。

**期望输出**：一张 3 列表（年份、等效分、对应当年位次），附 1-2 句趋势解读。

**第 2 轮｜专业方向初筛**  
> 孩子喜欢 **XX**、排斥 **YY**、不冷不热 **ZZ**。用 `plan_2026` + `score_phy2024/2025`，在等效分 ±20 分范围内，列出包含 XX 方向的所有院校专业组合。

**期望输出**：候选院校专业列表，每项含院校、专业、计划数、2024/2025 最低分与位次、所在地。

**第 3 轮｜冲稳保三档初稿**  
> 按等效分 ±15 划线，把第 2 轮的结果分到冲 / 稳 / 保三档，每档至少 8 个志愿。

**期望输出**：冲 / 稳 / 保三段，每段 ≥ 8 个志愿，每条含分数与位次。

**第 4 轮｜地理与家庭背景过滤**  
> 我家的硬约束：必须 **XX 省份**、学费 ≤ **X 万**、父母 **XX 行业** 可对接资源。请把第 3 轮清单按这些约束过滤，剔除不符合的。

**期望输出**：过滤后的精简志愿清单，附每条剔除原因。

**第 5 轮｜招生稳定性核查**  
> 把第 4 轮剩下的志愿逐一核查：`plan_2026` 计划数相比 2024 / 2025 是否有大幅变化？是否新设专业？是否停招？标记风险。

**期望输出**：每条志愿标注风险等级（低 / 中 / 高），含具体变化点。

**第 6 轮｜输出可打印志愿表**  
> 把最终志愿表整理成 Markdown 表格，字段：序号、院校、专业、计划数、2024 分 / 位次、2025 分 / 位次、冲稳保标记、备注（含风险标注）。

**最终输出**：一份可直接打印的志愿表 Markdown。

## 📌 数据范围与局限

- **覆盖年份**：2022–2026。
- **省份**：仅辽宁省。
- **已知局限**：
  - `plan_YYYY` 的次选科目（`second`）和招生计划数偶有更新滞后。
  - `score_his2022` 不存在（历史类录取分数表从 2023 年起）。
  - 本项目只提供数据与查询工具，**不替代**招生主管部门的官方信息。
- **使用建议**：最终填报以辽宁省教育考试院发布的官方数据为准。
