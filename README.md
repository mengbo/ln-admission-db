# ln-admission-db

辽宁省高考招生本地化数据库 + AI Agent 辅助报考工具包。

- 📊 **本地离线**：2022–2026 招生计划、录取分数、一分一段表打包进单个 SQLite 文件，零网络依赖。
- 🤖 **AI 辅助**：配合 [OpenCode](https://opencode.ai) / Claude Code 等 Agent，用自然语言提问即可自动跑 SQL、生成冲稳保清单。
- 📝 **问卷 → 方案**：填写 [templates/question.md](./templates/question.md) 一份问卷，AI 一次性按 [templates/report.md](./templates/report.md) 输出可打印的志愿填报方案。

![OpenCode 报考助力示例](https://github.com/mengbo/ln-admission-db/blob/main/assets/screenshot_oc.png?raw=true)

## 🎯 给家长看：这是啥、能干啥

高考志愿填报的核心问题只有两个：**我这个分能上啥学校？** **哪个学校哪个专业最适合我家孩子？**

本项目把辽宁省 5 年的招生计划、录取分数、一分一段表打包成一个 SQLite 数据库，再搭配 AI 助手（OpenCode / Claude Code）就能：

- 查分数对应的全省位次（一分一段表）
- 算今年分数相当于往年的多少分（等效分）
- 按"冲、稳、保"三档生成院校 + 专业清单
- 过滤地域、专业方向、升学规划等个性化条件
- 最终输出一份**完整的、可打印的志愿填报方案**

你不需要懂 SQL，只需要会跟 AI 助手说话就行。

## 🛠 给开发者看：背后原理

整个数据库围绕**三张表**展开：

| 表 | 作用 | 关键字段 |
|---|---|---|
| `plan_YYYY` | 院校专业的招生计划 | `inst`, `major`, `plan_num`, `first`, `second` |
| `score_phyYYYY` / `score_hisYYYY` | 院校专业的最低录取分 | `inst`, `major`, `min_score` |
| `rank_phyYYYY` / `rank_hisYYYY` | 一分一段表（分数→位次） | `score`, `rank` |

核心查询模式：

```sql
-- 2024 物理类 620 分对应的位次
SELECT score, rank FROM rank_phy2024 WHERE score = 620;

-- 2024 物理类 605~635 分的院校专业
SELECT s.inst, s.major, s.min_score, r.rank
FROM plan_2024 p
JOIN score_phy2024 s ON p.inst_code = s.inst_code AND p.major_code = s.major_code
JOIN rank_phy2024    r ON s.min_score = r.score
WHERE p.first LIKE '%物%' AND s.min_score BETWEEN 605 AND 635
ORDER BY s.min_score DESC;
```

详细字段说明见 [docs/db_schema.md](./docs/db_schema.md)。

## 🤖 AI Agent 辅助报考：3 步上手

> 强烈推荐 [OpenCode](https://opencode.ai) 或 Claude Code：本地 shell 即可调 `sqlite3`，零配置。  
> 不推荐使用 MCP 协议方案（Cline / Cherry Studio），相关文档仅作历史参考：[docs/MCP.md](./docs/MCP.md)。

1. 在项目根目录启动 OpenCode / Claude Code。
2. 让 Agent 执行 `./scripts/createdb.sh` 构建数据库。
3. 打开 [templates/question.md](./templates/question.md) 填写考生信息，复制到对话发给 AI，按 [templates/report.md](./templates/report.md) 输出志愿填报方案。

详细步骤见下一节「📋 问卷 → 方案：完整工作流」。

## 📋 问卷 → 方案：完整工作流

### 第 1 步：打开问卷

打开 [templates/question.md](./templates/question.md)，里面是一份简洁的填空式问卷：

```
考生信息
分数（必填）：
首选科目（物理/历史 缺省物理）：
选科组合（如 化学+生物 必填）：
想学专业（缺省不限）：
不想学的专业（缺省无）：
升学或就业方向（缺省未定；常见：本科就业/读研/出国/考公/央国企）：
学校位置（缺省 600+ 全国不限，600- 倾向辽宁周边；可填具体省份/城市）：
家庭就业资源（缺省无；可写父母/亲属行业可对接的资源）：
其他（缺省跳过；可写身体条件/特殊要求等）：
```

### 第 2 步：填写考生信息

按行填上对应信息，留空 = 用括号里的缺省值。**必填项** 分数、首选科目、选科组合必须填。

填好后的示例（考生 2026 物理类、620 分）：

```
考生信息
分数（必填）：620
首选科目（物理/历史 缺省物理）：物理
选科组合（如 化学+生物 必填）：化学+生物
想学专业（缺省不限）：计算机、电子信息
不想学的专业（缺省无）：
升学或就业方向（缺省未定；常见：本科就业/读研/出国/考公/央国企）：读研
学校位置（缺省 600+ 全国不限，600- 倾向辽宁周边；可填具体省份/城市）：
家庭就业资源（缺省无；可写父母/亲属行业可对接的资源）：
其他（缺省跳过；可写身体条件/特殊要求等）：
```

### 第 3 步：发给 AI，触发方案

把上述内容复制到对话中，并明确告诉 AI 按方案模板输出：

> 请按 [templates/report.md](./templates/report.md) 模板，生成我孩子的完整志愿填报方案。

### 第 4 步：AI 输出 5 节方案

AI 收到后会自动按方案模板输出 5 节内容：

| 节 | 内容 | 用途 |
|---|---|---|
| 方案头 | 考生信息回显 + 核心数据 | 完整复述 question.md 填写内容 + 今年分数/位次 |
| 1 | 定位分析 | 今年分数 → 位次 → 去年/前年等效分 |
| 2 | 冲稳保志愿清单 | 每档（冲/稳/保）独立做"去年 15 + 前年 15 → 综合去重"，不强凑固定数量 |
| 3 | 风险与备注 | 计划数变化、新设/停招专业、风险等级 |
| 4 | 最终志愿表 | 可直接打印的志愿清单（按 (院校, 专业) 全局去重） |
| 5 | 院校简要介绍 | 所有院校的城市/历史/优势专业 |

**关键方法**（在 report.md 中详细说明）：
- 方案与具体年份解耦，框架内 `今年/去年/前年` 三层表，AI 自动适配考生高考年份
- 冲稳保**每档独立**用去年、前年的实际录取数据筛选（每档"去年 15 + 前年 15"），合并后按 (院校, 专业) 去重——不强凑固定数量，多几个少几个都没关系
- 三档交叉很正常（同一个志愿可能因去年/前年分差跨档），不强行重新归类，最终志愿表全局去重即可
- 适配辽宁 100+ 平行志愿容量

### 第 5 步：复核与精调

- 看完方案后，对有疑问的志愿可在对话中追问 AI（如"详细分析 XX 大学 XX 专业"）
- 对风险等级高的志愿，可让 AI 提供替代选项
- 最终填报以辽宁省教育考试院发布的官方数据为准

### 方案存档

AI 输出的方案建议保存到本地 `reports/` 目录（已被 `.gitignore` 忽略，仅作本地存档）。**命名约定以时间戳为准**，详见 [AGENTS.md 方案存档约定](./AGENTS.md#-方案存档约定)。

## 🔧 单点提问（可选）

> 如果你只想问**一个具体问题**（比如"等效分多少分"或"某专业有哪些学校"），直接复制下方任一条发给 AI 即可。
> 想跑完整分析请用上面的 [templates/](./templates/) 流程。

### 模板 1：等效分换算

> 我家孩子 2026 年辽宁物理类考了 **XXX 分**，请帮我：
> 1. 查 2026 物理类一分一段表，得到位次。
> 2. 用这个位次，分别回查 2025、2024 物理类一分一段表，得到等效分。
> 3. 整理成表格输出。

**期望输出**：一张 3 列表（年份、等效分、与今年的分差），含趋势解读。

### 模板 2：冲稳保院校清单

> 我家孩子 2026 物理类 **XXX 分、位次 XXX**，请生成冲稳保三档院校清单：
> - 冲：等效分 + 15 分以上
> - 稳：等效分 ± 15 分
> - 保：等效分 - 15 分以下
>
> 每条标注：院校名、专业名、计划数、近两年最低分与位次、所在地。

**期望输出**：冲 / 稳 / 保三段，每段一张表（按分数降序）。

### 模板 3：单所院校深度分析

> 请深度分析 **XXX 大学** 在辽宁物理类近 3 年（2024 / 2025 / 2026）的招生数据：
> 1. 哪些专业在辽宁招生、各专业计划数。
> 2. 各专业近两年最低录取分与对应位次。
> 3. 是否有新设专业、停招专业、计划数大幅变化。
> 4. 在我家孩子位次附近的录取概率评估。

**期望输出**：4 段分析 + 关键数据表 + 概率结论（冲 / 稳 / 保）。

### 模板 4：单类专业横向对比

> 我家孩子想读 **计算机 / 电子信息** 类，请列出辽宁物理类 2026 年所有招收此类专业的院校：
> 1. 院校层次（985 / 211 / 双一流 / 普通本科）。
> 2. 各院校此专业的计划数。
> 3. 各院校此专业 2024–2025 最低分与位次。
> 4. 按位次排序，标注稳 / 冲建议。

**期望输出**：一张大表（按 2025 位次升序）+ 前 5 / 后 5 简要点评。

## 📌 数据范围与局限

- **覆盖年份**：2022–2026。
- **省份**：仅辽宁省。
- **已知局限**：
  - `plan_YYYY` 的次选科目（`second`）和招生计划数偶有更新滞后。
  - `score_his2022` 不存在（历史类录取分数表从 2023 年起）。
  - 本项目只提供数据与查询工具，**不替代**招生主管部门的官方信息。
- **使用建议**：最终填报以辽宁省教育考试院发布的官方数据为准。
